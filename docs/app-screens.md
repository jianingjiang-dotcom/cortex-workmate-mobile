## 对话（Workmate / Agents）

### Workmate 对话主界面

![Workmate 对话主界面](./screens/05_Workmate%E5%AF%B9%E8%AF%9D%E4%B8%BB%E7%95%8C%E9%9D%A2.png)

- **定位**：产品核心 —— 与 Workmate 的一条**连续、不分 session**的对话流。
- **元素**：顶部 [收藏] · [Workmate / Agents 分段] · [搜索][通知]；消息流（含工具调用卡 / 审批卡 / MCP 连接卡）；底部输入框（附件 / 全屏编辑）。
- **交互**：流式回复（思考 → 工具调用 → 正文）；回复下方 复制 / 赞 / 踩 / 收藏。

### Workmate 历史搜索

![Workmate 历史搜索](./screens/06_Workmate%E5%8E%86%E5%8F%B2%E6%90%9C%E7%B4%A2.png)

- **定位**：在连续对话流里按关键词检索历史消息。
- **交互**：输入即时过滤；点结果跳回该消息并高亮 1.8s。

### 收藏中心

![收藏中心](./screens/07_Workmate%E6%94%B6%E8%97%8F%E4%B8%AD%E5%BF%83.png)

- **定位**：集中查看收藏的 Workmate 回复（按收藏时间倒序）。
- **交互**：点条目跳回原对话并高亮；行尾实心书签取消收藏；支持搜索。

### 通知跳回 · 对话内授权审批卡

![通知跳回 · 对话内授权审批卡](./screens/08_%E9%80%9A%E7%9F%A5%E8%B7%B3%E5%9B%9E-%E5%AE%A1%E6%89%B9%E5%8D%A1.png)

- **定位**：点「授权请求」通知 → 跳回 Workmate 对话内的审批卡（带高亮框，外框与图标留有安全间距）。
- **元素**：审批卡**第一级（折叠态）即可操作**：拒绝 / 同意（同意默认授予 7 天有效期，已同意态展示「已同意 · 有效期 7 天」）；展开仅追加函数与参数。
- **交互**：同意 / 拒绝后卡片状态与通知中心药丸同步更新。

### 通知跳回 · 对话内 MCP 连接卡

![通知跳回 · 对话内 MCP 连接卡](./screens/09_%E9%80%9A%E7%9F%A5%E8%B7%B3%E5%9B%9E-%E8%BF%9E%E6%8E%A5%E5%8D%A1.png)

- **定位**：点「连接请求」通知 → 跳回对话内的 MCP 连接卡，在原上下文完成 OAuth 连接。
- **元素**：连接卡（服务 Logo + 名称 + 描述 + 「连接」按钮）；连接成功后变绿色「已连接」并继续任务。

### Agents 对话主界面

![Agents 对话主界面](./screens/10_Agents%E5%AF%B9%E8%AF%9D%E4%B8%BB%E7%95%8C%E9%9D%A2.png)

- **定位**：与 Workmate 连续流互补的**多智能体对话**，按会话 / 项目组织。
- **元素**：顶部 [历史] · 分段 · [通知]；输入框上方 **Agent 栏**（左：当前智能体选择 pill；右：MCP 服务器 pill + 已启用计数）；助手回复带智能体署名。
- **交互**：可切换智能体 / 快捷管理 MCP；用户消息可编辑重发。

### 选择智能体

![选择智能体](./screens/11_%E6%99%BA%E8%83%BD%E4%BD%93%E9%80%89%E6%8B%A9%E5%99%A8.png)

- **元素**：底部弹窗「选择智能体」+ 搜索；按组织分组（内置智能体 / Cobo / Sales & BD / Operation & Growth / RD / Default）；行 = 头像 + 名称 + 描述，当前项打勾。
- **交互**：点选即切换（toast 提示），后续回复以该智能体署名。

### 编辑任务

![编辑任务](./screens/17_%E5%AE%9A%E6%97%B6%E4%BB%BB%E5%8A%A1%E7%BC%96%E8%BE%91.png)

- **元素**：名称 / 指令 / 触发时机三模式：周期重复（周内多选 + 每天快捷）· 指定日期（多选具体日期时间，各运行一次）· 每隔一段时间（N 分钟/小时/天，可设开始时间）。
- **交互**：未改动「保存」不可点（dirty 门控）；空名保存时回退原名称。

## 助手 · 会议记录

### 会议记录列表

![会议记录列表](./screens/20_%E4%BC%9A%E8%AE%AE%E8%AE%B0%E5%BD%95%E5%88%97%E8%A1%A8.png)

- **元素**：按 今天 / 昨天 / 更早 分组；行 = 来源图标（录音/导入配色区分）+ 标题 + 时间·时长 + 状态药丸（由单一 status 驱动：转译中=蓝色药丸内嵌进度填充 / 转译失败=红色外露 / 未转写=中性灰，已完成无药丸）；底部居中「开始录制」；右上「上传音频」。
- **行为**：命名弹窗含「保存后转译」多选（默认勾选）；勾选则保存/导入后进入统一转译（上传+转写）显示「转译中」、完成变已完成；任一阶段失败=红色「转译失败」，详情「重试转译」。

### 会议详情 · 逐字稿

![会议详情 · 逐字稿](./screens/21_%E4%BC%9A%E8%AE%AE%E8%AF%A6%E6%83%85-%E9%80%90%E5%AD%97%E7%A8%BF.png)

- **元素**：常驻播放器（进度 / ±15s / 倍速）；转写 | 总结 分段；说话人着色的逐字稿，点击可定位播放。
- **交互**：右上搜索可检索转写内容；⋯ 菜单含 重新转译 / 复制全文 / 重命名 / 删除。

### 会议详情 · AI 总结（模板 chip + 更新时间）

![会议详情 · AI 总结（模板 chip + 更新时间）](./screens/22_%E4%BC%9A%E8%AE%AE%E8%AF%A6%E6%83%85-%E6%80%BB%E7%BB%93.png)

- **元素**：总结卡上方显示**当前模板 chip**（如 会议纪要）与「总结更新于 …」时间；Markdown 总结 + 复制按钮。
- **说明**：时间戳让「仅重新生成总结」也有可见变化。

### 转译 · 选择总结模板（含总结备注）

![转译 · 选择总结模板（含总结备注）](./screens/23_%E8%BD%AC%E8%AF%91%E6%A8%A1%E6%9D%BF%E9%80%89%E6%8B%A9.png)

- **定位**：点「转译」先选总结模板，不同模板 = 不同的总结内容格式。
- **元素**：5 个模板单选（会议纪要 / 客户总结 / 面试记录 / 通用 / 行动项清单，各带一行描述，默认按标题智能推荐）；**「总结备注（可选）」**输入框（给 AI 补背景，会以“背景补充”写进总结）；底部「开始转译」。

### 重新转译（重选模板 + 可选重生成逐字稿）

![重新转译（重选模板 + 可选重生成逐字稿）](./screens/24_%E9%87%8D%E6%96%B0%E8%BD%AC%E8%AF%91.png)

- **定位**：已完成的会议可重跑：**总结一定重新生成**；逐字稿是否重生成由开关决定（默认关）。
- **元素**：模板单选（预选当前模板）+「同时重新生成逐字稿」开关 + hint + 总结备注（回填上次）+「开始」。
- **交互**：仅总结 → toast「已更新总结」；含逐字稿 → toast「已重新生成逐字稿与总结」。

### 录音中

![录音中](./screens/25_%E5%BD%95%E9%9F%B3%E9%A1%B5.png)

- **元素**：滚动镜像波形 + 大号计时器；中间 暂停/继续 大圆钮 + 右侧「结束」。
- **交互**：结束 → 命名弹窗（含「保存后转译」多选，默认勾选）→ 保存；勾选则回列表显示「转译中」，不勾选则「未转写」。

## 通知中心

### 通知中心（按时间聚合）

![通知中心（按时间聚合）](./screens/26_%E9%80%9A%E7%9F%A5%E4%B8%AD%E5%BF%83.png)

- **定位**：三类通知 —— 定时任务状态 / Tool Call 审批 / MCP 连接请求，各自点击跳到处理页。

- **说明**：状态变化只改药丸、**不改标题**；审批 / 连接跳回 Workmate 对话内卡片完成。
