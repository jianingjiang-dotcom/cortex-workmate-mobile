# Cortex Workmate · 移动端高保真原型

> 一版 **iOS 风格、纯 mock 数据、全流程可点通** 的移动端交互原型，依据 `Cortex-Workmate-PRD.md` 从零搭建。
> 在桌面浏览器中以「手机设备外框」承载展示。

## 运行

```bash
pnpm install
pnpm dev
```

打开终端提示的地址（默认 http://localhost:5173 ）。建议把浏览器窗口调高一些，手机外框会自动缩放适配。

生产构建：`pnpm build` → `pnpm preview`

## 技术栈

- **React + Vite + TypeScript**
- **Tailwind CSS**（iOS 设计语言：大标题、分段控件、底部 Tab、底部 Sheet、列表分组、轻阴影、安全区）
- **Zustand + persist**（用户创建/修改 **本地持久保留**，刷新仍在；支持「重置演示数据」）
- **framer-motion**（流式打字、声纹波形、转场、Sheet）
- **react-markdown + remark-gfm**（AI 回复 / 会议总结的 Markdown，含勾选项）

无任何真实后端：登录、大模型、转写均为 **预置脚本 / 假数据 / 模拟动画**。

## 已实现模块（对应 PRD 第 6 节）

- **登录与开屏引导**：Google 登录（加载态）→ 首次进入必经 3 页轮播引导（可跳过）→ 主界面；退出回登录。
- **AI 对话**：顶部 `[Workmate | 普通对话]` 分段切换。
  - Workmate 模式：连续、无会话，进入即见历史，可「加载更早」。
  - 普通对话：历史抽屉（项目分组 + 未分组）、会话新建/重命名/删除/移动、对回复重试、对自己消息编辑重发。
  - 通用能力：**自定义附件选择器**（应用内底部面板：示例照片网格 + 文件列表，纯 mock、不调用系统选择）、语音输入（波形动画 → 落入输入框）、**打字机流式（工具调用卡 → 答复，回复带 Workmate 名称标头）**、停止生成、复制、点赞/点踩（可选原因）。
  - **Workmate 设置**：点顶栏左上角 Workmate 头像进入（名称/描述/头像/系统提示词/**模型**，本地保留）；模型已并入此处，不再单独切换。
- **通知**：任意 Tab 右上角常驻铃铛 + 未读角标；通知中心含「工具调用审批」（用途 + 权限范围 + 授权有效期 1天/7天/永不/自定义 + 同意/拒绝）与「任务状态变更」；**对话内审批卡与通知中心实时同步**；任务通知可跳转任务 / 触发的对话。
- **定时任务**：**仅经对话用自然语言创建**（可解析→建任务并联动出现在面板 + 通知；不可解析→反问澄清）。列表（状态/计划/来源/上次·下次运行、行内 立即运行·暂停·删除、无新建按钮、空态引导去对话）。详情（概览 + 指令全文 + 运行记录）。立即运行 / 暂停恢复 / 编辑（改触发时机与描述，重算下次运行）/ 删除。**运行 ↔ 对话联动**：每次执行生成一条带「由定时任务「X」于 某时间 触发」来源横幅的对话。
- **会议记录**：录音（**双向镜像柱状声纹波形** + 计时 + 停止/取消 + 麦克风权限说明与降级）、导入本地音频、命名入列表；**手动触发转译**（分阶段等待态）；详情播放器（播放/暂停/进度拖动/±15 秒/倍速/当前·总时长）、**按说话人分离的转写**（不同颜色、播放时高亮跟随并滚动、点句跳转）、Markdown 总结；重命名/删除；失败态 + 重试。
- **我的**：账户（头像/名称/邮箱）、**语言中/英即时切换**、外观、版本、**重置演示数据**（二次确认）、**退出登录**（二次确认）。

## 端到端可演示路径

1. 登录 → 3 页引导 → 对话（Workmate 默认高亮）。
2. 对话发「**每天早上 9 点帮我整理 GitHub PR**」→ 见思考 + 工具调用卡 + 「已创建定时任务」卡 → 助手 → 定时任务里出现该任务（角标 +1）。
3. 对话发「**查看我日历上的今日安排**」→ 出现对话内审批卡（用途 + 权限 + 有效期）→ 在卡片或通知中心同意 → 两处状态同步、对话追加结果。
4. 助手 → 定时任务 → 任意任务详情 → 立即运行 → 运行记录可「打开对话」（带触发来源横幅）。
5. 助手 → 会议记录 → 录音（波形+计时）或对「待分析」录音点「转译」（等待）→ 打开「产品评审会」边播边看高亮转写 + 总结。
6. 右上铃铛 → 通知中心审批 / 任务动态跳转。
7. 我的 → 中英切换即时生效；重置演示数据恢复初始；退出登录回登录页。

## 说明

- **全局状态**：各模块均含空 / 加载 / 错误 / 权限态；删除、退出、重置等不可逆操作均二次确认；操作有 toast 反馈。
- **持久化**：数据存于浏览器 localStorage（键 `cortex-workmate-v1`）。「重置演示数据」恢复初始示例。
- **演示诚实性**：整体即声明为演示原型；内容均为预置示例。
- 已生成的对话内容（历史消息、转写、总结）保留其生成时的语言；所有 **界面文案** 支持中/英即时切换，默认中文。
